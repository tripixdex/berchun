# Docs Index

Каталог `docs` разложен по смысловым группам:

- `project/` — базовая спецификация, аудит входов, карта входных данных и открытые вопросы.
- `governance/` — общие правила работы и верхнеуровневая дорожная карта.
- `report/` — контракты и планы, относящиеся к teacher-facing report surface.
- `delivery/` — delivery/export surface, output formats и связанные матрицы.
- `ux/` — operator-facing UX и one-button сценарии.
- `methodical/architecture/` — архитектура, контракты, шаблоны и декомпозиция методички.
- `methodical/content/` — текущие guide-source документы.
- `methodical/microops/` — самый низкий уровень методической декомпозиции.

Если нужен актуальный guide source, смотри:

- `docs/methodical/content/METHODICAL_GUIDE.md`
- `docs/methodical/content/METHODICAL_GUIDE_GENERAL_SOURCE.md`
